DEV announces eight winners across two categories of the Hermes Agent Challenge. Build With Hermes Agent winners include an offline-first bushfire response platform, a 25-agent Polish Parliament that drafts bills with legal citations, a system converting prompts into model weights, and an AI debate judge; Write About Hermes Agent winners cover agent design philosophy, setup guides, and framework critiques. Each of the eight winners received $125 USD, DEV++ Membership, and an Exclusive DEV Badge.
